# Notepad IDE 🚀

A modern, high-performance, and privacy-focused code editor built for the web.

![Notepad IDE Preview](https://raw.githubusercontent.com/ivanroldannoson/notepad/main/public/vite.svg)

## 🌟 Key Features

- **Multi-Tab Workspace**: Manage multiple files simultaneously with an intuitive tabbed interface. Switch between documents seamlessly without losing progress.
- **Professional IDE Layout**: A distraction-free, full-screen workspace with hidden gutters and a clean UI inspired by modern code editors.
- **Theme Suite**: Switch between **Dark** and **Light** modes instantly. The interface and the code editor sync perfectly to match your preference.
- **Smart Code Formatting**: Pre-installed support for formatting JavaScript, TypeScript, HTML, CSS, SQL, JSON, Python, and more. Keep your code clean with a single click (`Shift+Alt+F`).
- **Language Support**: Syntax highlighting for over 10 common programming languages, powered by the industry-standard Ace Editor.
- **Native File Handling**: Open local files from your computer and save your work directly back to your local storage.

## 🔒 Privacy & Data Policy

**Your data belongs to you.**

Notepad IDE is a client-side application. We do not have servers, and **we do not store any of your data**. 
- All files you open or create remain on your local machine.
- Your session (open tabs, unsaved content, and theme preferences) is stored locally in your browser's `localStorage`.
- No tracking, no cookies, and no telemetry.

## 🛠️ Technology Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS for a premium, responsive UI.
- **Editor Core**: [Ace Editor](https://ace.c9.io/)
- **Icons**: Lucide React
- **Automation**: GitHub Actions for CI/CD and automated deployment.
