# Notepad IDE 🚀

A modern, high-performance, and privacy-focused code editor built for the web. Designed for developers who value speed, clean aesthetics, and complete data ownership.

![Notepad IDE Preview](https://raw.githubusercontent.com/ivanroldannoson/notepad/main/public/vite.svg)

## 🎨 Premium Experience

Notepad IDE offers a professional-grade workspace with **vibrant themes**, **smooth animations**, and **Apple-style typography**, all while being completely **serverless and open-source**.

## 🌟 Key Features

- **Multi-Tab Workspace**: Manage multiple files with an intuitive tabbed interface. Features **drag-and-drop reordering** and **dirty-state indicators**.
- **Global Search (`⇧⌘F`)**: High-speed search across all open tabs with line-level navigation and snippet previews.
- **Sidebar Explorer**: A collapsible side panel to manage open editors and access common tools. Displays search and settings in its collapsed "Activity Bar".
- **Zen Mode (`⌘K Z`)**: A distraction-free environment that hides all UI chrome, letting you focus solely on your code.
- **Auto-save**: Never lose work again. Enable it in settings to automatically persist changes to local storage.
- **Integrated Breadcrumbs**: Real-time navigation showing your file path and language context.
- **Side-by-Side Diff View**: Compare any two open tabs with synchronized scrolling and line-level changes.
- **Smart Code Formatting**: Built-in support for Prettier, JS-Beautify, and SQL-Formatter. Quick format with `⇧⌥F`.
- **Advanced Language Detection**: Automatically detects language based on file extension or content. Support for 20+ languages.
- **Professional Export**: Export your code as a high-quality **Image** or a text-only **PDF** for documentation and sharing.

## 🔒 Privacy & Data Policy

**Your data belongs to you.**

Notepad IDE is a client-side application. We do not have servers, and **we do not store any of your data**. 
- All files remain on your local machine.
- Your session (open tabs, unsaved content, and theme preferences) is stored locally in your browser's `localStorage`.
- No tracking, no cookies, and no telemetry.

## 🛠️ Technology Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS (Premium UI)
- **Editor Core**: [Ace Editor](https://ace.c9.io/)
- **Icons**: Lucide React
- **Deployment**: Automated GitHub Actions to `gh-pages`.
